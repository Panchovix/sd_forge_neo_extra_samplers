# Extra samplers (from Comfy and reForge) for Forge Neo.

This extension add some samplers that are on reForge/ComfyUI, but aren't on forge Neo for different reasons.

Any sampler can be added, but let me know if you want one in specific.

Added samplers for now:

* Gradient Estimation
* Gradient Estimation CFG++
* SEEDS 2
* SEEDS 3
* DPM++ SDE CFG++
* SA Solver
* SA Solver PECE
